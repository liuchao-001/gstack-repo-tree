# Product

Who gstack is for, what it promises, and the strategic choices that shape
every skill.

gstack is a suite of opinionated Claude Code skills that turn a single
builder into a virtual engineering team. The product thesis is that AI
collapses the "20-person team" into "one person + agents" — but only if the
workflow is opinionated enough to replace real roles (CEO, Eng Manager,
Designer, Reviewer, QA, Security, Release) with structured slash commands
that talk to each other.

This domain captures the **decisions** behind that thesis — what belongs
in V1, what is deliberately out, and the philosophical commitments that
newer skills must continue to honor. If a skill proposal contradicts a
decision here, the proposal needs a conversation, not a PR.
